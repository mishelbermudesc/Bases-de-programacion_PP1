# Bases-de-programacion_PP1
Este proyecto automatiza el proceso de carga, limpieza, análisis y visualización de un conjunto de datos climáticos mediante el uso de bucles y funciones, para mejorar la eficiencia y la reproducibilidad del análisis de la información.

# Proyecto: Automatización de Análisis y Visualización de Datos Climáticos

El proyecto incluye la carga y preparación de datos, revisión y limpieza básica, análisis exploratorio mediante funciones, creación de nuevas columnas derivadas, agrupamientos con Pandas y generación de gráficos con Matplotlib y Seaborn.

## Contenido del repositorio

- `Proyecto_Clima_CORREGIDO`: notebook principal donde se desarrolla el análisis paso a paso.
- `Weather_Data-1.csv`: base de datos climática utilizada en el proyecto.
- Gráficos y resultados generados durante el análisis exploratorio.
- Archivo modificado/exportado con nuevas columnas derivadas, como la sensación térmica (Solo 50.000 porque no me dejó completar la ejecución cuando intenté guardar el archivo con el millón de datos).

## Herramientas utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objetivo del proyecto

Automatizar el análisis de datos climáticos mediante el uso de funciones, estructuras de datos, agrupamientos, filtros y visualizaciones, con el fin de obtener resultados organizados y facilitar la interpretación de las variables meteorológicas por ciudad y por periodo de tiempo.
